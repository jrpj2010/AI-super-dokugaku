### コア機能
- **簡単入力**: 一言二言でOK
- **3D強化**: X軸・Y軸・Z軸での多角的強化
- **視覚化**: 3Dビジュアライザーで変化を体感
- **AI実行**: ワンクリックでAI連携

- **フロントエンド**: Next.js 14 (App Router) + TypeScript
- **スタイリング**: Tailwind CSS + shadcn/ui
- **バックエンド**: Server Actions + Supabase
- **AI連携**: OpenAI / Anthropic / Google AI
- **インフラ**: Google Cloud Run + Docker
