JSONファイルを読み込むためのローカルサーバー起動方法

なぜローカルサーバーが必要？

ブラウザのセキュリティ制限により、file://プロトコルではJSONファイルを読み込めません。ローカルサーバーを起動することで、http://プロトコル経由でファイルにアクセスできるようになります。

方法1: Python（推奨）

ターミナルを開き、このフォルダに移動してから以下のコマンドを実行：

python3 -m http.server 8000

その後、ブラウザで http://localhost:8000/index.html を開く

方法2: Node.js（npxを使用）

Node.jsがインストールされている場合：

npx http-server -p 8000

その後、ブラウザで http://localhost:8000/index.html を開く

方法3: VS Code拡張機能

VS Codeを使用している場合、「Live Server」拡張機能をインストールして、index.htmlを右クリックし「Open with Live Server」を選択

サーバーの停止方法

ターミナルで Ctrl + C（Mac: Command + C）を押す
